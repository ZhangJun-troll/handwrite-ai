# ✍️ HandWrite AI — 手写对话应用方案

## 产品概念
在 iPhone 上用手写字体与 AI 对话。你用手写输入问题，AI 用手写体风格回复。

## 技术架构

### 方案A：原生 Swift App（推荐上架 App Store）
```
┌─────────────────────────────────────────┐
│  iPhone App (SwiftUI + PencilKit)       │
│                                         │
│  ┌───────────┐  ┌───────────────────┐   │
│  │ 手写画布   │  │ AI 回复区（手写体）│   │
│  │ PencilKit  │  │ 自定义字体渲染     │   │
│  │ 手指/笔    │  │                    │   │
│  └─────┬─────┘  └────────▲──────────┘   │
│        │                  │              │
│        ▼                  │              │
│  ┌──────────┐    ┌────────┴──────────┐  │
│  │ VisionKit│    │   LLM API 调用     │  │
│  │ OCR识别   │───▶│ (OpenAI/自定义)    │  │
│  └──────────┘    └───────────────────┘  │
└─────────────────────────────────────────┘
```

### 方案B：Web App（最快验证，可转PWA/巨魔/自签）
```
iPhone Safari/WebView
  ├── Canvas 手写画布（touch事件）
  ├── 手写识别 → Tesseract.js / Google Vision API / Apple VisionKit（WKWebView可用）
  ├── AI API 调用 → 任意 OpenAI 兼容接口
  └── 回复渲染 → 手写字体（如 "Ma Shan Zheng" 思源手写体）
```

## 核心模块

### 1. 手写输入
- **方案A**: `PencilKit` — Apple原生，体验最好，支持压感
- **方案B**: `<canvas>` + touch事件 — 跨平台，可用手指或笔

### 2. 手写识别（笔迹→文字）
| 方案 | 精度 | 离线 | 成本 | 推荐 |
|------|------|------|------|------|
| Apple VisionKit (VNRecognizeTextRequest) | ⭐⭐⭐⭐⭐ | ✅ | 免费 | iOS原生首选 |
| Tesseract.js | ⭐⭐⭐ | ✅ | 免费 | Web方案 |
| Google Cloud Vision | ⭐⭐⭐⭐⭐ | ❌ | 付费 | 最准但要钱 |
| 自己的视觉API | ⭐⭐⭐⭐ | ❌ | API费 | 用你已有的NVIDIA NIM |

**推荐**: 用你已有的 NVIDIA NIM key（nemotron-nano-12b-v2-vl），免费且精度不错

### 3. AI 回复
- API地址：`https://integrate.api.nvidia.com/v1/chat/completions`
- 模型：`nvidia/nemotron-nano-12b-v2-vl`（免费）或 DeepSeek/Qwen
- System prompt: 让AI用简洁、口语化的方式回答，模拟手写笔记风格

### 4. 手写体渲染（文字→手写外观）
- **方案A**: 使用 Google Fonts 手写字体（Ma Shan Zheng、ZCOOL XiaoWei、Liu Jian Mao Cao）
- **方案B**: 随机化字符间距、大小、轻微旋转，模拟手写不规则感
- **方案C**: 用 Canvas 逐字渲染，每个字随机偏移+倾斜

## 推荐路线

### 快速验证（1-2天）→ Web App
1. 写一个单文件 HTML（Canvas手写 + 识别 + AI + 手写体渲染）
2. 部署到你的 Cloudflare Pages
3. iPhone 添加到主屏幕（PWA）
4. 测试效果

### 正式产品（1-2周）→ Swift App
1. SwiftUI + PencilKit 手写输入
2. Apple VisionKit 离线手写识别（免费、精度最高）
3. 调用你的 API（NVIDIA NIM / DeepSeek / 自定义）
4. 自定义手写体字体渲染
5. 打包上架 App Store
